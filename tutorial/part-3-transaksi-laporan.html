📦 Konsep Transaksi Barang

Di sistem inventaris, ada 2 jenis transaksi utama:

✅ Barang MASUK

  • Barang datang dari supplier
  • Stok BERTAMBAH
  • Formula: stok = stok + jumlah
  • Contoh: Beli 10 laptop → stok +10

❌ Barang KELUAR

  • Barang dijual/dipakai
  • Stok BERKURANG
  • Formula: stok = stok - jumlah
  • ⚠️ Stok tidak boleh minus!

Contoh Kasus Sederhana

Laptop ASUS stoknya 10 unit. Ada 5 unit datang dari supplier (barang masuk), stok jadi 15. Lalu customer beli 3 unit (barang keluar), stok jadi 12. Kalau customer mau beli 20 unit? TOLAK! Karena stok cuma 12.

1 Barang Masuk (app/transaksi/masuk.php)

File ini untuk mencatat barang yang masuk ke gudang. Ingat: Proses form SEBELUM include header!

app/transaksi/masuk.php PHP 
<?php
/**
 * Transaksi Barang Masuk
 * Proses form SEBELUM include header!
 */

session_start();
date_default_timezone_set('Asia/Jakarta');

// Cek login
if (!isset($_SESSION['user_id'])) {
    header('Location: /bahan-ajar-ukk/app/auth/login.php');
    exit;
}

require_once __DIR__ . '/../config/koneksi.php';

// Ambil daftar barang untuk dropdown
$stmt = $pdo->query("SELECT id, nama_barang, stok FROM barang ORDER BY nama_barang");
$daftar_barang = $stmt->fetchAll();

$error = '';

// PROSES FORM SEBELUM OUTPUT HTML
if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $id_barang = (int) ($_POST['id_barang'] ?? 0);
    $jumlah = (int) ($_POST['jumlah'] ?? 0);
    $keterangan = trim($_POST['keterangan'] ?? '');
    
    // Validasi input
    if ($id_barang <= 0) {
        $error = 'Pilih barang terlebih dahulu!';
    } elseif ($jumlah <= 0) {
        $error = 'Jumlah harus lebih dari 0!';
    } else {
        try {
            // Mulai transaction
            $pdo->beginTransaction();
            
            // 1. INSERT ke tabel transaksi
            $stmt = $pdo->prepare("INSERT INTO transaksi (id_barang, jenis_transaksi, jumlah, keterangan) VALUES (?, 'masuk', ?, ?)");
            $stmt->execute([$id_barang, $jumlah, $keterangan]);
            
            // 2. UPDATE stok barang (TAMBAH)
            $stmt = $pdo->prepare("UPDATE barang SET stok = stok + ? WHERE id = ?");
            $stmt->execute([$jumlah, $id_barang]);
            
            // Commit - simpan permanen
            $pdo->commit();
            
            $_SESSION['flash_message'] = "Barang masuk berhasil!";
            $_SESSION['flash_type'] = 'success';
            header('Location: riwayat.php');
            exit;
            
        } catch (Exception $e) {
            // Rollback - batalkan semua kalau error
            $pdo->rollback();
            $error = 'Terjadi kesalahan!';
        }
    }
}

// BARU include header setelah proses selesai
$page_title = 'Barang Masuk';
require_once __DIR__ . '/../includes/header.php';
?>

<!-- HTML Form di sini -->
<div class="page-header">
    <h1>Barang Masuk</h1>
</div>

<div class="card">
    <div class="card-body">
        <?php if ($error): ?>
            <div class="alert alert-danger"><?= htmlspecialchars($error) ?></div>
        <?php endif; ?>
        
        <form method="POST">
            <div class="mb-3">
                <label>Pilih Barang *</label>
                <select name="id_barang" class="form-select" required>
                    <option value="">-- Pilih --</option>
                    <?php foreach ($daftar_barang as $b): ?>
                        <option value="<?= $b['id'] ?>"><?= htmlspecialchars($b['nama_barang']) ?> (Stok: <?= $b['stok'] ?>)</option>
                    <?php endforeach; ?>
                </select>
            </div>
            <div class="mb-3">
                <label>Jumlah Masuk *</label>
                <input type="number" name="jumlah" class="form-control" min="1" required>
            </div>
            <div class="mb-3">
                <label>Keterangan</label>
                <textarea name="keterangan" class="form-control"></textarea>
            </div>
            <button type="submit" class="btn btn-success">Simpan</button>
        </form>
    </div>
</div>

<?php require_once __DIR__ . '/../includes/footer.php'; ?>

Penjelasan Kode Penting

$pdo->beginTransaction() = Mulai transaction. Semua query setelah ini belum disimpan permanen.
$pdo->commit() = Simpan semua query secara permanen ke database.
$pdo->rollback() = Batalkan semua query kalau ada yang error.
stok = stok + ? = Menambah stok. Tanda ? akan diisi oleh nilai $jumlah.

Kenapa Pakai Transaction?

Bayangkan: INSERT transaksi berhasil, tapi UPDATE stok gagal. Data jadi tidak sinkron! Dengan transaction, kalau ada 1 yang gagal, semuanya dibatalkan. Data tetap konsisten.

2 Barang Keluar dengan Validasi Stok

Sama seperti barang masuk, tapi ada validasi stok karena stok tidak boleh minus!

app/transaksi/keluar.php (bagian validasi) PHP 
// PROSES FORM SEBELUM OUTPUT HTML
if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $id_barang = (int) ($_POST['id_barang'] ?? 0);
    $jumlah = (int) ($_POST['jumlah'] ?? 0);
    
    // CEK STOK DULU - Ini yang beda dari barang masuk!
    $stmt = $pdo->prepare("SELECT stok, nama_barang FROM barang WHERE id = ?");
    $stmt->execute([$id_barang]);
    $barang = $stmt->fetch();
    
    if (!$barang) {
        $error = 'Barang tidak ditemukan!';
    } elseif ($jumlah <= 0) {
        $error = 'Jumlah harus lebih dari 0!';
    } elseif ($barang['stok'] < $jumlah) {
        // VALIDASI STOK - Tidak boleh keluar lebih dari stok yang ada!
        $error = "Stok tidak cukup! Stok {$barang['nama_barang']} hanya {$barang['stok']}";
    } else {
        try {
            $pdo->beginTransaction();
            
            // INSERT transaksi dengan jenis 'keluar'
            $stmt = $pdo->prepare("INSERT INTO transaksi (id_barang, jenis_transaksi, jumlah, keterangan) VALUES (?, 'keluar', ?, ?)");
            $stmt->execute([$id_barang, $jumlah, $keterangan]);
            
            // UPDATE stok (KURANGI, bukan tambah!)
            $stmt = $pdo->prepare("UPDATE barang SET stok = stok - ? WHERE id = ?");
            $stmt->execute([$jumlah, $id_barang]);
            
            $pdo->commit();
            
            $_SESSION['flash_message'] = "Barang keluar berhasil!";
            header('Location: riwayat.php');
            exit;
            
        } catch (Exception $e) {
            $pdo->rollback();
            $error = 'Terjadi kesalahan!';
        }
    }
}

PENTING: Validasi Stok!

Sebelum barang keluar, WAJIB cek: $barang['stok'] < $jumlah

Kalau stok tidak cukup, TOLAK transaksi. Ini mencegah stok menjadi minus.

Perbedaan Barang Masuk vs Keluar

Aspek Barang Masuk Barang Keluar
Jenis transaksi 'masuk' 'keluar'
Update Stok stok + jumlah stok - jumlah
Validasi Stok Tidak perlu WAJIB

3 Riwayat Transaksi dengan DataTables

Menampilkan semua transaksi dengan fitur search, filter, dan pagination menggunakan DataTables.

app/transaksi/riwayat.php PHP 
<?php
$page_title = 'Riwayat Transaksi';
require_once __DIR__ . '/../includes/header.php';
require_once __DIR__ . '/../config/koneksi.php';

// Ambil filter dari URL
$filter_jenis = $_GET['jenis'] ?? '';
$filter_dari = $_GET['dari'] ?? date('Y-m-01');  // Default: awal bulan
$filter_sampai = $_GET['sampai'] ?? date('Y-m-d'); // Default: hari ini

// Query dengan JOIN untuk ambil nama barang
$sql = "SELECT t.*, b.nama_barang 
        FROM transaksi t 
        JOIN barang b ON t.id_barang = b.id 
        WHERE DATE(t.tanggal_transaksi) BETWEEN ? AND ?";
$params = [$filter_dari, $filter_sampai];

// Kalau ada filter jenis, tambahkan kondisi
if ($filter_jenis) {
    $sql .= " AND t.jenis_transaksi = ?";
    $params[] = $filter_jenis;
}

$sql .= " ORDER BY t.tanggal_transaksi DESC";

$stmt = $pdo->prepare($sql);
$stmt->execute($params);
$data_transaksi = $stmt->fetchAll();
?>

<div class="page-header">
    <h1>Riwayat Transaksi</h1>
</div>

<!-- Form Filter -->
<div class="card mb-4">
    <div class="card-body">
        <form method="GET" class="row g-3">
            <div class="col-md-3">
                <label>Jenis</label>
                <select name="jenis" class="form-select">
                    <option value="">Semua</option>
                    <option value="masuk" <?= $filter_jenis === 'masuk' ? 'selected' : '' ?>>Masuk</option>
                    <option value="keluar" <?= $filter_jenis === 'keluar' ? 'selected' : '' ?>>Keluar</option>
                </select>
            </div>
            <div class="col-md-3">
                <label>Dari</label>
                <input type="date" name="dari" class="form-control" value="<?= $filter_dari ?>">
            </div>
            <div class="col-md-3">
                <label>Sampai</label>
                <input type="date" name="sampai" class="form-control" value="<?= $filter_sampai ?>">
            </div>
            <div class="col-md-3 d-flex align-items-end">
                <button type="submit" class="btn btn-primary w-100">Filter</button>
            </div>
        </form>
    </div>
</div>

<!-- Tabel -->
<div class="card">
    <div class="card-body">
        <table id="tabelTransaksi" class="table table-hover">
            <thead>
                <tr>
                    <th>No</th>
                    <th>Tanggal</th>
                    <th>Barang</th>
                    <th>Jenis</th>
                    <th>Jumlah</th>
                </tr>
            </thead>
            <tbody>
                <?php $no = 1; foreach ($data_transaksi as $t): ?>
                <tr>
                    <td><?= $no++ ?></td>
                    <td><?= date('d/m/Y H:i', strtotime($t['tanggal_transaksi'])) ?></td>
                    <td><?= htmlspecialchars($t['nama_barang']) ?></td>
                    <td>
                        <span class="badge <?= $t['jenis_transaksi'] === 'masuk' ? 'bg-success' : 'bg-danger' ?>">
                            <?= ucfirst($t['jenis_transaksi']) ?>
                        </span>
                    </td>
                    <td class="<?= $t['jenis_transaksi'] === 'masuk' ? 'text-success' : 'text-danger' ?> fw-bold">
                        <?= $t['jenis_transaksi'] === 'masuk' ? '+' : '-' ?><?= $t['jumlah'] ?>
                    </td>
                </tr>
                <?php endforeach; ?>
            </tbody>
        </table>
    </div>
</div>

<?php require_once __DIR__ . '/../includes/footer.php'; ?>

<!-- DataTables - HARUS SETELAH footer! -->
<script>

Penjelasan Kode

JOIN barang b ON t.id_barang = b.id = Menggabungkan tabel transaksi dengan tabel barang untuk mendapatkan nama_barang.
$_GET['jenis'] = Mengambil parameter dari URL. Form dengan method="GET" mengirim data lewat URL.
date('Y-m-01') = Tanggal hari pertama bulan ini.
$('#tabelTransaksi').DataTable() = Mengaktifkan plugin DataTables pada tabel.

Ingat! Script DataTables SETELAH Footer

Script DataTables harus diletakkan SETELAH include footer.php karena jQuery dimuat di footer!

4 Export Laporan ke Excel (CSV)

Cara export data ke Excel tanpa library tambahan (menggunakan CSV):

app/laporan/export_excel.php PHP 
<?php
session_start();
require_once __DIR__ . '/../config/koneksi.php';

// Cek login
if (!isset($_SESSION['user_id'])) {
    header('Location: ../auth/login.php');
    exit;
}

// Ambil data
$stmt = $pdo->query("SELECT t.*, b.nama_barang FROM transaksi t JOIN barang b ON t.id_barang = b.id ORDER BY t.tanggal_transaksi DESC");
$data = $stmt->fetchAll();

// Set header untuk download
header('Content-Type: text/csv; charset=utf-8');
header('Content-Disposition: attachment; filename=laporan_' . date('Y-m-d') . '.csv');

// Buat file untuk output
$output = fopen('php://output', 'w');

// BOM untuk Excel bisa baca UTF-8
fprintf($output, chr(0xEF) . chr(0xBB) . chr(0xBF));

// Header kolom
fputcsv($output, ['No', 'Tanggal', 'Barang', 'Jenis', 'Jumlah', 'Keterangan']);

// Data
$no = 1;
foreach ($data as $row) {
    fputcsv($output, [
        $no++,
        date('d/m/Y H:i', strtotime($row['tanggal_transaksi'])),
        $row['nama_barang'],
        ucfirst($row['jenis_transaksi']),
        $row['jumlah'],
        $row['keterangan'] ?? '-'
    ]);
}

fclose($output);
exit;

Penjelasan Export CSV

header('Content-Type: text/csv') = Memberitahu browser bahwa ini file CSV.
header('Content-Disposition: attachment') = Memaksa browser untuk download, bukan tampilkan.
fopen('php://output', 'w') = Membuat "file" langsung ke output browser.
fputcsv() = Menulis baris data dalam format CSV.
