Yo! 👋 Selamat datang di gudang ilmu persiapan **Ujian Kompetensi Keahlian (UKK)**! Ini adalah tempat dimana mimpi-mimpi kamu untuk lulus ujian dimulai (atau setidaknya tempat kamu belajar sebelum tidur di jam 2 pagi). 😅

Berisi tutorial, materi pembelajaran, kuis, dan simulasi yang bakal bikin otak kamu dipakai secara maksimal!

## 🎯 Cara Pake Project Ini (Jangan Bikin Sendiri Caranya!)

1. **Mulai dari materi** - Baca `materi-ukk.html` dan `materi-git.html` dulu biar gak bingung di tutorial
2. **Ikutin tutorial** - Selesain 3 part tutorial ini biar sistem kamu jadi (dan jangan copas doang 👀)
3. **Main di kuis** - Kerjain `kuis.html` dan `kuis-lengkap.html` sampe score bagus
4. **Simulasi** - Kerjain `simulasi-ukk.html` berkali-kali biar confident pas ujian asli

## 💡 Tips Biar Lulus (Verified Tips™)

✅ Buat catatan sambil belajar (gak usah nulis semuanya, poinnya aja)  
✅ Selesain ketiga tutorial **berurutan** (jangan lompat-lompat)  
✅ Kalau stuck, baca ulang materi (sering ada jawaban di situ)  
✅ Kerjain kuis berkali-kali sampe mahir (target score 80+!)  
✅ Simulasi ujian beberapa kali biar gak gugup pas ujian beneran  
✅ Istirahat yang cukup (otak butuh rechargein kawan 🧠)
